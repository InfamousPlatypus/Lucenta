# Lucenta Roadmap

## Phase 1: Core Foundation (Current)
- [x] **Local-First Architecture**: Core Triage and Model Management.
- [x] **MCP Integration**: Dynamic server loading and tool discovery.
- [x] **Tiered Model Manager**: Complexity-based routing with 'Step-Up' logic.
- [x] **Autonomous Deep Research**: Plan -> Reflect -> Act -> Synthesize workflow.
- [x] **Research & Deep Research Modes**: Dual intensity settings with context.
- [x] **Structured Markdown Report System**: Automatic saving of exhaustive reports.
- [x] **Human-in-the-Loop (HIL)**: Safety-first Python execution.
- [x] **Memory System**: Implicit learning and contextual retention.
- [ ] **Multi-modal Support**: Vision and Audio ingestion.

## Phase 2: Domain Specialization
- [ ] **Code Specialist**: Advanced local coding assistant with project-wide awareness.
- [ ] **Security Layer 2**: Real-time auditing of all tool outputs and code.
- [ ] **Edge Deployment**: Optimized builds for mobile/single-board NPUs.

## Phase 3: Collective Intelligence
- [ ] **Multi-Agent Swarms**: Collaborative problem solving.
- [ ] **Decentralized Memory**: P2P knowledge sharing between instances.
